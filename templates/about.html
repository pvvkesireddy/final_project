About This Project

This project utilizes machine learning to detect and analyze cyberbullying and sentiment within textual content. Below is an outline of how it works:

Features

  • Cyberbullying Detection: Uses a pre-trained machine learning model to classify text as cyberbullying or non-cyberbullying.
  • Sentiment Analysis: Analyzes the sentiment of the text to determine if the content is positive, neutral, or negative.
  • Latest News: Provides updates and news on cyber crimes, helping users stay informed about recent developments.

Technology Stack

The application is built using Flask, a Python web framework, with machine learning models implemented using libraries such as scikit-learn and nltk for natural language processing.
